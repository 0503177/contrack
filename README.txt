Instructions to load / view

1) Git Clone project into a folder
2) Using terminal, go to that folder
3) type this without quotes, "npm install package-lock.json"
4) Wait for it to finish
5) If port 80 is already in use on your computer, go into the last line of server.js and change it.
6) In terminal, type this without quotes, "npm run devStart"
7) Connect to site.

Login is non functional. So to access bulk of the material you will need to type in ip:port/mainmenu
